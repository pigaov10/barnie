**My Other Themes (Just $4):** http://wrapbootstrap.com/user/ashobiz

**Features:**

- Minimal Admin theme.
- Fast loading, Simple and Clean.
- No unnecessary fonts. Uses system base font.
- Uses LESS (for colors).
- Simple sidebar (responsive).
- Mini header with dropdown menus for notifications.
- Gritter plugin for notification.
- Charts with tab support on home page.
- jQuery knob chart.
- Widgets with customizable header background color. 
- Widgets with scroll bar support.
- Task management widget.
- Contacts widget.
- Project management widget.
- Pages widget.
- Chats widget.
- Report widget. 
- Simple documentation.
- 404 Error page.
- Calendar page with google calendar support.
- Error log page to list the errors generated by your site.
- Form validate support.
- Form wizard support.
- Gallery page to list your photos.
- Simple email page with inbox, read mail and compose mail pages.
